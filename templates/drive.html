On This Page You Will Learn About Different Types Of Memory Drives

Memory Drives are where users save their saved data

Unlike RAM the memory saved on memory drives are not temporary

Most commonly used memory drives in computers are Hard Disk Drives HDD and Solid State Drives SSD

HDD use a spinning disk withhin it to process the stored data

SSD use intergrated circuts within it to process the stored data

SDD comes in two factors: M.2 and U.2

U.2 is similar to the HDD the way it is connected

Both require SATA power to be powered and SATA data cable to be plugged into the motherboard

M.2 SSD looks similar to a RAM stick however it is plugged into the NVME slot of a motherboard
