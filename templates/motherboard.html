On This Page You Will Learn About Motherboards

A motherboard has various different names. Most notables are main board and main circut board.
Its functions allow communications between other hardware components.
The boards are designed specifically to function with specific components, so not all components will work.
Motherboards are usually located on the left or right panel of a computer depending with its case.
Motherboards are built with various components. Each component function are explained in the other links.
Most notable components are:
PCI slots
CPU sockets
3pin fan connectors
4pin power connectors
24pin ATX power connectors
CPU socket
Memory slots
Jumpers
1394 Connectors
