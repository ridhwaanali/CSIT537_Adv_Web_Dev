On This Page You Will Learn About RAM

RAM, or Random Access Memory, are memory sticks that are placed on the memory slots on a Motherboard

Ram is responsible for handeling various different memory addresses from applications and memory stored inside a computer

Each row handles various different addresses

The memory addresses are saved within the rows however it is temporary

Depending on the motherboard some may allow single chanel Ram or dual chanel meaning one stick or two sticks

Also depending on the motherboard Ram can be oriented in slot 0 and 2 or slot 1 and 3
